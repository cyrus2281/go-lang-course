Introduction

Go (or Golang) is an open-source programming language created at Google in 2007. It combines the performance of a compiled language with the simplicity of a modern one. Programs run fast, compile almost instantly, and scale well — which is why Go powers projects like Kubernetes, Docker, and Terraform.

Code examples

Every Go snippet has a Copy button and a Run button. Click Run and an embedded Go Playground appears below the snippet — the code is already on your clipboard, paste it with ⌘/Ctrl+V and click Run inside the playground.

package main

import "fmt"

func main() {
    // The canonical first program.
    fmt.Println("Hello, Go!")

    // Loops and slices:
    nums := []int{1, 2, 3, 4, 5}
    sum := 0
    for _, n := range nums {
        sum += n
    }
    fmt.Printf("sum of %v = %d\n", nums, sum)
}

Shell commands use the bash language and get copy buttons without the Run option:

# Create a new Go module and run your first program
mkdir learn-go && cd learn-go
go mod init example.com/learn-go
echo 'package main

import "fmt"

func main() { fmt.Println("Hello, Go!") }' > main.go
go run .

Common pitfalls

!
Forgetting package main Every runnable program needs exactly one file with package main and a main() function. Library packages use any other name.
⚠
Unused imports and variables Go refuses to compile code with unused imports or unused local variables. Use the blank identifier _ or remove the offender.

Practice exercises

EXERCISE 1

Print a greeting for each argument

Write a program that iterates over os.Args[1:] and prints "Hello, NAME!" for each argument. If no arguments are given, print "Hello, stranger!".

Hints: import os and fmt. Use for _, name := range .... Check len(os.Args) == 1 for the no-arg case.

Show one possible solution 
package main

import (
    "fmt"
    "os"
)

func main() {
    args := os.Args[1:]
    if len(args) == 0 {
        fmt.Println("Hello, stranger!")
        return
    }
    for _, name := range args {
        fmt.Printf("Hello, %s!\n", name)
    }
}
